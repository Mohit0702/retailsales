# retailsales
RetailPulse — Sales Insights Dashboard

A Python-based sales analysis project that cleans multi-region retail data, computes KPIs, and generates a full visual dashboard using Matplotlib and Seaborn.
📊 Dataset


1,000 transactions across 2023
3 categories — Beauty, Clothing, Electronics
5 price tiers — ₹25, ₹30, ₹50, ₹300, ₹500
Features: year, month, category, quantity, unit price, revenue



🔍 Key Findings


Electronics drove the highest revenue despite fewer orders — high unit price effect
May was the best performing month; September the weakest
Teams choosing ₹500 price tier generated the most total revenue
Clothing had the highest order volume but lower avg order value than Electronics
Automated data cleaning pipeline reduced manual preprocessing effort significantly

▶️ How to Run

bash# 1. Install dependencies
pip install pandas numpy matplotlib seaborn

# 2. Run the analysis script
python retailpulse_charts.py

# 3. Open the dashboard
# Open retailpulse_dashboard.html in any browser

Running the script will:


Load and clean retailpulse_sales.csv
Compute KPIs (total revenue, orders, units, avg order value)
Generate all 6 charts and save them to the charts/ folder
Print a summary of key findings in the terminal
